# Citadel Skill 用户记忆文件说明

本文件是 **格式规范文档**，描述 citadel skill 在用户本地维护的记忆文件的结构约定。

**实际记忆文件路径**：`~/.cache/oa-skills/citadel-memory.md`

该文件不随 skill 分发，存在于用户本地，由 skill 在用户首次触发记忆写入时自动创建和维护。

---

## 文件格式约定

记忆文件是一个普通 Markdown 文件，以二级标题（`##`）划分场景章节。每个章节的内容是该场景的当前偏好值，没有偏好时内容为 `（暂无偏好）`。

**读写原则**：
- 每次修改只改目标章节，其他章节内容保持不变。
- 章节标题即为场景的唯一 key，新增场景时在文件末尾追加新二级标题章节即可。
- 不要删除章节标题本身，无偏好时将内容替换为 `（暂无偏好）`。

---

## 初始模板

文件不存在时，首次写入前创建以下内容：

```markdown
# Citadel Skill 用户偏好记忆

> 此文件由 citadel skill 自动维护。
> 文件路径：~/.cache/oa-skills/citadel-memory.md

## 创建文档默认位置

（暂无偏好）

## 常用文档模板

（暂无模板）

## 文档默认密级

（暂无偏好）
```

---

## 建议场景的写入格式

以下是当前已使用的场景示例，供 AI 写入时参考格式。新场景可自由扩展，不受限于以下列表。

### ## 创建文档默认位置

记录用户希望默认存放新文档的目录。写入格式：

```markdown
## 创建文档默认位置

- label：《周会记录》
- parentId：2756933117
- 备注：用户于 2026-05-19 确认
```

清除时将内容替换为 `（暂无偏好）`。

---

### ## 常用文档模板

记录用户保存的模板别名与 templateId 的映射。写入格式（每条一行，可有多条）：

```markdown
## 常用文档模板

- 别名：周报模板 | templateId：2751442505 | 备注：每周五用
- 别名：复盘模板 | templateId：2751336167
```

删除某条时移除对应行；全部删除时内容替换为 `（暂无模板）`。

---

### ## 文档默认密级

记录用户希望新文档默认使用的密级。写入格式：

```markdown
## 文档默认密级

- label：C3（内部敏感）
- level：3
```

清除时将内容替换为 `（暂无偏好）`。
