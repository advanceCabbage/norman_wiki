# 📈 daily_stock_analysis

> 基于 AI 大模型的自动化股票分析平台，每日自动抓取多市场行情、生成「决策仪表盘」报告并推送到企业微信/飞书/Telegram 等渠道。

🌐 **仓库**：[ZhuLinsen/daily_stock_analysis](https://github.com/ZhuLinsen/daily_stock_analysis)

---

## 核心功能

- **多市场覆盖**：A 股、港股、美股、日股、韩股的行情、技术指标、资金流向、新闻
- **AI 决策报告**：核心结论、评分、买卖点位、风险警报、催化因素
- **自动推送**：企业微信、飞书、Telegram、Discord、Slack、邮件
- **零成本部署**：可用 GitHub Actions 免费定时运行，无需服务器

## 技术栈

Python · TypeScript · 支持 OpenAI/Claude/DeepSeek 等 LLM · AkShare/Tushare/YFinance 数据源
