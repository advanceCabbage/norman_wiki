# 🔤 OpenAI Tokenizer

> OpenAI 官方的在线分词工具：把一段文本拆成 GPT 模型实际看到的 token，并实时显示 token 数与字符数。

🌐 **地址**：[platform.openai.com/tokenizer](https://platform.openai.com/tokenizer)

---

## 一句话介绍

输入任意文本，它会用彩色高亮标出每个 token 的切分方式，并给出 token 总数——直观理解「模型眼里的文本」与估算 API 成本的小工具。
