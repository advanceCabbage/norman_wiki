| **W2** | RAG 系统化 | chunking、embedding 选型/微调、混合多路召回、重排序、Lost-in-Middle、RAG 评估 | 向量数据库 · RAG 与知识检索 · GraphRAG | 项目 A 接**评估 + trace**，跑出召回率/准确率 |
| **W3** | Agent 核心 | Agent vs Chain、ReAct/CoT/ToT 取舍、Function Calling 原理、Memory | AI Agent 简介/核心组件/工作原理 · 推理与规划 · Agent 架构 · 第一个 AI Agent · 工具调用与函数调用 · 记忆系统设计 · LangChain/LangGraph/CrewAI | 启动项目 B：LangGraph 多 Agent + 工具调用 |
